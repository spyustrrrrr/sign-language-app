# 🤟 Belajar Bahasa Isyarat

Sebuah aplikasi web interaktif untuk belajar bahasa isyarat (SIBI/ASL) dari huruf A-Z, angka, dan kata-kata dasar. Dibangun dengan MERN stack (MongoDB, Express, React, Node.js).

## ✨ Fitur

- 📖 **Belajar Huruf A-Z** - Pelajari bentuk tangan untuk setiap huruf
- 🔢 **Belajar Angka 1-10** - Angka dasar dalam bahasa isyarat
- 💬 **Kata Dasar** - Kata-kata yang sering digunakan sehari-hari
- 🖼️ **Media Interaktif** - Gambar dan video demonstrasi
- 📱 **Responsive Design** - Bisa diakses dari HP, tablet, atau laptop
- 🔍 **Filter Kategori** - Pilih kategori sesuai kebutuhan belajar

## 🛠️ Tech Stack

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - Database
- **Cloudinary** - Media storage (gambar/video)

### Frontend
- **React.js** - UI library
- **Vite** - Build tool
- **TailwindCSS** - Styling
- **Axios** - HTTP client
